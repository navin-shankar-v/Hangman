# Hangman Game

A classic word-guessing game built with React and Vite.

## How to Play

1. Choose a category: **Movies** or **Words**
2. Guess letters one at a time to reveal the hidden word
3. You have **7 chances** to guess incorrectly before the game ends
4. Guess the complete word before running out of attempts to win!

## Features

- Two categories to choose from: Movies and Words
- Visual hangman stage progression
- Interactive letter selection
- Win/lose game states with answer reveal
